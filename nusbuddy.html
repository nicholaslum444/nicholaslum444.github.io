I created this app with Chong Jia Wei as part of Orbital 2014.
Visit our Project Blog.
View our Github repository.


Installation:

Copy the APK file to your Android phone then install it from your phone's file browser.

Please enable "Install from unknown/external sources" in your phone's Settings -> Security.

Login:

Use your IVLE username and password to log in. Omit the domain name from the username.

Example

UserID: a0123456

Password: ivlepassword123
